一、政策概述

  1. 本政策旨在说明您使用 “小星记账”（以下简称 “本应用”）时，我们对个人信息的处理规则，保障您的信息安全。
  2. 本应用为本地运行软件：无账号注册 / 登录体系，您的所有数据仅存储于设备本地，我们不会将其上传至任何服务器。
  3. 本应用集成第三方 SDK 以实现统计、支付等功能（详见第四章），相关 SDK 可能收集必要信息，我们已审核其合规性，您可通过链接查看 SDK 隐私政策。
  4. 除本政策另有规定外，未征得您事先许可，我们不会对外披露或向第三方提供您的个人信息。

 

二、个人信息的收集与使用

我们仅在本地收集以下与服务相关的信息，无服务器上传行为，所有信息均用于实现对应功能：

1. 设备信息

我们的产品集成友盟SDK，用于统计与分析小星记账的产品运营数据（例如新增用户与活跃用户数量）。友盟SDK需要收集您的设备Mac地址、唯一设备识别码（Android ID/IDFA/IDFV/OAID/OpenUDID/GUID）以提供统计分析服务。

2. 应用安装信息与运行中的进程信息

为了正常调用微信、支付宝的登录、分享与支付功能，我们需要获取您的手机应用安装信息或运行中的进程信息来判断您是否有安装微信、支付宝应用并进行页面跳转或提示。

3. 剪切板信息

我们的自动记账功能中提供了短信识别功能，其中包括「读取剪切板中的短信」选项。当您开启此选项后，在进入应用主页时，我们将读取一次剪切板数据，用于识别您已复制的短信内容。采集频率为：每次进入应用首页时采集一次。

在记账界面以及账户详情页中，长按金额/账户余额区域，会出现「复制|粘贴」选项。当点击粘贴按钮时，我们将读取一次剪切板数据，用于快捷修改金额数据。采集频率为：每点击一次粘贴按钮采集一次。

4. 应用内请求获取的权限
  • 文件读写权限

    在您使用数据备份与恢复、账单导入与导出或从相册中选择账单图片等功能时，由于要将数据备份（导出）至外部存储，或从外部存储恢复（导入）数据，小星记账会向您请求文件读写权限。该权限的获取仅用来实现相关功能，不会读写其他与功能无关的文件。

  • 录音权限

    在您使用语音记账功能时，由于要解析您的语音内容完成语音记账操作，小星记账会向您请求录音权限。

  • 相机权限

    在您使用拍照功能添加账单图片时，小星记账会向您请求相机权限。

  • 定位权限

    在您使用保存地理位置信息功能时，由于需要在每次记账完成后自动获取当前地理位置信息，小星记账会在您开启此功能选项时向您请求定位权限。

  • 无障碍权限

    在您使用自动记账功能时，需要主动开启无障碍服务开关。自动记账的实现原理是通过系统的无障碍服务，在用户主动打开特定应用的特定界面时，自动获取一次界面数据。数据经过本地算法识别整理后，通过悬浮窗展示在用户面前，方便用户一键记账。自动记账获取的界面数据仅用于实现该功能，且识别算法均为本地算法，获取的界面数据不会上传网络。

  • 短信接收权限

    在您使用自动记账-短信识别功能时，由于需要获取系统新接收的银行短信信息，需要您主动授予应用短信接收权限。短信接收和识别操作仅在当前手机内部完成，短信内容不会上传网络。

 

三、信息存储与注销

1. 存储方式

  • 应用数据：所有记账数据、微信 / 支付宝授权信息、剪切板 / 短信 / 界面识别数据均仅存储于您的设备本地，存储路径为应用沙盒内。

  • 备份数据：

    • 本地备份：存储于您设备根目录【Documents】下的【小星记账】文件夹
    • 云盘备份：存储于您个人云盘内的【小星记账备份】文件夹

  • 无云端同步 / 服务器备份：我们不会存储或上传您的任何数据至服务器，您需通过【数据备份与恢复】功能自行管理备份文件，避免数据丢失。

2. 注销说明

小星记账是一款本地软件，无需注册和登录账号，您的所有数据仅存储在设备本地，在您清除应用数据或卸载应用后，所有数据也将随之删除，不用额外进行注销操作。

  • 删除您授权过的微信/支付宝账号信息

    您在产品激活界面授权的微信/支付宝账号信息仅存储在应用本地，用于生成和验证产品密钥。在您清除应用数据或卸载应用后，您授权的账号信息也将随之删除。您亦可通过【激活界面-右上角菜单-注销账号】来单独删除已授权的账号信息。除了删除应用内的账号信息，您还可以在微信和支付宝的应用中解除对小星记账的授权，操作路径如下：

    • 微信：我 - 设置 - 个人信息与权限 - 授权管理 - 小星记账 - 解除授权
    • 支付宝：我的 - 右上角设置按钮 - 隐私 - 授权管理 - 小星记账 - 解除授权
  • 删除您的备份数据

    如果您使用过数据备份功能，您可以自行决定是否删除本地和云盘中的备份文件。删除备份文件后，未来将无法通过备份文件恢复记账数据：

    • 本地：若使用过本地备份，可打开手机的文件管理器，删除【Documents】目录下的【小星记账】文件夹
    • 云盘：若使用过云盘备份，可登录您的个人云盘并删除其中的【小星记账备份】文件夹

五、自启动与关联启动说明

1. 桌面小组件：Android Widget

小星记账内为您提供了多个可供选择的Android桌面小组件（小部件），方便您在桌面上快捷浏览账本数据。但由于其可能存在自启动或者被系统关联启动的情况，特此说明如下：

  • 方式：小星记账的桌面小组件必须由您手动添加到手机桌面才能显示，应用不会自动添加。
  • 目的：方便用户在桌面上快捷地进行记账、打开应用等操作，并可以查看账本中的统计数据。
  • 范围：除小星记账本身外，桌面小组件不会与其他应用进行通信。
  • 提示：当您手动添加小组件到桌面后，小组件会被系统不定期激活，但小组件仅限于本地数据的读取和显示。如果您添加的是 MIUI 小部件，小部件将会以单独的 Widget 进程启动，与小星记账的主进程进行通信，此为 MIUI 小部件的特殊机制。

如果您不希望小星记账的桌面小组件进行自启动或被关联启动，可在桌面长按小组件进行移除。

2. 广播接收器：SmsBroadcastReceiver

小星记账提供了银行短信自动识别功能，方便您在收到银行短信时进行自动记账。因为需要监听系统短信的接收情况，我们向系统注册了一个短信广播接收器，在系统收到新短信时会向小星记账发送广播提醒。由于广播接收器可能存在自启动或者被系统关联启动的情况，特此说明如下：

  • 方式：在小星记账主页侧边栏-自动记账中开启短信识别，并授予小星记账短信接收权限。
  • 目的：方便用户根据银行短信内的金额变动信息进行自动记账。
  • 范围：除小星记账本身外，短信广播接收器不会与其他应用进行通信。

如果您不希望小星记账的短信广播接收器SmsBroadcastReceiver进行自启动或被关联启动，可在自动记账主页中关闭短信识别开关，并在权限管理中关闭小星记账的短信接收权限。

六、信息安全保障

  1. 小星记账的所有数据均存储在您的设备本地。备份数据时，备份文件会经过加密处理。
  2. 注意：您需要通过应用内【数据备份与恢复】功能备份您的记账数据，并自行保管好备份文件，避免数据丢失。关于【数据备份与恢复】功能的详细信息可通过本应用内【主页侧边栏 - 帮助与反馈】查看。
